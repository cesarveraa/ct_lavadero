Paquete de íconos para Victoria Lavadero.

Incluye:
- favicon.ico
- favicon-16x16.png
- favicon-32x32.png
- favicon-48x48.png
- apple-touch-icon.png
- android-chrome-192x192.png
- android-chrome-512x512.png
- mstile-150x150.png
- og-image.png
- site.webmanifest
- head-snippet.html

Uso:
1. Copia estos archivos a la carpeta public/ de tu proyecto.
2. Usa el snippet del archivo head-snippet.html en el <head>.
3. Limpia caché del navegador al probar.

Nota:
El logo original es horizontal, así que se adaptó a formato cuadrado para verse mejor como favicon.
